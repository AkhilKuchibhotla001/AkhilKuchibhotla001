## 🔬 Featured Showcase Projects

### 👁️ [Deep Learning Retinal Classification Portfolio](https://github.com/AkhilKuchibhotla001/Deep-Learning-Project)
An advanced multiclass diagnostic framework designed to identify **9 pathological and healthy conditions** from fundus photography scans. 
*   **Key Highlights**: Integrated Custom **Squeeze-and-Excitation (SE)**, **CBAM**, and parameter-free **SimAM** attention layers into PyTorch backbones.
*   **Result**: Achieved a peak testing accuracy of **77.22%** using an attention-calibrated ResNet50 model, outperforming standard DenseNet, ConvNeXt, and Vision Transformer (ViT-B/16) baselines.
*   **Visual Reports**: Features detailed clinical definitions, Mermaid pipeline flowcharts, confusion matrices, and training curves.
